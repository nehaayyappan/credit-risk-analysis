# Credit Risk Analysis
This project analyzes borrower characteristics to predict loan default risk using data-driven methods.

## Methods
- Exploratory Data Analysis (EDA)
- Data Visualization
- Logistic Regression
- Random Forest

## Key Findings
- Loan-to-income ratio is the strongest predictor of default
- Prior default history significantly increases risk
- Higher income and longer employment reduce default risk

## Files
- `eda_portfolio.ipynb` – Data exploration and visualizations  
- `final_portfolio.ipynb` – Modeling and final analysis  

## Tools
Python, pandas, seaborn, matplotlib, scikit-learn
